## 💬 구현 예정
+ `요구사항1` 
  + 서로 다른 두개의 값이 입력 되었을 때 다른 값을 확인하여 화면에 표기 기능 구현
+ `요구사항2`
  + Firebase의 cloud firestore를 세팅하여 데이터를 저장하고 불러오기 기능 구현
+ `요구사항3`
  + 다양한 경우의 수를 고려하여 가능한 가장 높은 수준의 데이터 비교 기능 구현
